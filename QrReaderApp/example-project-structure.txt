Key Files to Create First:
===========================

1. MauiProgram.cs - Configure DI container
2. Services/Interfaces/IQrCodeService.cs - Define service contract
3. Services/Implementations/QrCodeService.cs - Implement QR scanning logic
4. ViewModels/BaseViewModel.cs - Base class for all VMs
5. ViewModels/QrScannerViewModel.cs - Scanner page VM
6. Views/QrScannerPage.xaml - Scanner UI
7. Platform-specific permission configurations

Recommended Creation Order:
===========================

Step 1: Project Setup
- Create MAUI project
- Install NuGet packages
- Configure MauiProgram.cs

Step 2: Models & Interfaces
- Create model classes
- Define service interfaces

Step 3: Services
- Implement services
- Register in DI container

Step 4: ViewModels
- Create BaseViewModel
- Implement feature ViewModels

Step 5: Views
- Create XAML pages
- Wire up data bindings

Step 6: Platform Configuration
- Add permissions
- Test on devices
